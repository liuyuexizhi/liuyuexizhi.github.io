# ntpd

> 目标：搭建生产可用的主从节点的时间同步服务器

1. 安装 ntpd、ntpdate

   ​`yum install -y ntpd ntpdate`

2. 主节点配置

   cat /etc/ntp.conf

   ```yaml
   server cn.ntp.org.cn             iburst  prefer
   server ntp.aliyun.com            iburst
   server ntp.tuna.tsinghua.edu.cn  iburst

   logfile /var/log/ntp.log
   ```

   解释：

   - iburst => 大概是立即同步一次&&加快同步时间的作用

   - prefer => 优先选择该服务器同步
3. 从节点配置

   cat /etc/ntp.conf

   ```yaml
   server 10.200.1.103              iburst  prefer
   server cn.ntp.org.cn             iburst
   server ntp.aliyun.com            iburst
   server ntp.tuna.tsinghua.edu.cn  iburst

   logfile /var/log/ntp.log
   ```

   解释：

   - 优先同步主节点时间服务器
4. 启动

   ```yaml
   systemctl start ntpd
   systemctl enable ntpd
   ```
